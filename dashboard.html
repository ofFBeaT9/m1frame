🚀 Quick Start — Use m1frame for a Project
1
Reset & set your project
Clear old data and describe your project in purpose.md
python scripts/reset_workspace.py
Then edit purpose.md with your research questions.
2
Feed your documents (PDFs)
Drop PDFs into the ingest panel below, or run:
python scripts/ingest_pdf.py your_doc.pdf "topic hint"
Each PDF becomes a structured wiki source page.
3
Run the 7-pillar pipeline
State your project goal — all 7 agents run automatically:
python -m m1frame --goal "Your research goal"
Add --parallel --stream for speed.
4
Query your knowledge base
Ask questions against ingested wiki pages:
python -c " from llm_client import LLMClient, load_config from agents.wiki import LLMWiki cfg = load_config('config.yaml') wiki = LLMWiki(LLMClient(cfg)) print(wiki.query('your question here')) "
5
Start the REST API (optional)
HTTP endpoint for programmatic access & webhooks:
pip install fastapi uvicorn httpx python api/server.py
Then POST to http://localhost:8080/run
